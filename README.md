# BankAppAbp
Bu proje, **Piton Teknoloji** stajım süresince **Mustafa Doğan** mentörlüğünde geliştirdiğim ilk ABP (ASP.NET Boilerplate) projemdir.
## Hakkında
Staj sürecinde ABP Framework kullanılarak geliştirilmiş bir banka uygulamasıdır. Proje kapsamında .NET ekosistemi, katmanlı mimari yapısı ve ABP Framework'ün temel özellikleri öğrenilmiş ve uygulanmıştır.
